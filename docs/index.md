# ufw firewall

**ufw** makes it easier to *add rule* in in your **iptalbes** to *allow ssh* for incoming connectins.

###### *Install* ufw

`sudo apt install ufw`

###### *Enable* ufw

`sudo ufw enable`

###### Check ufw *status*

`sudo ufw status verbose`

###### Add rule to *allow SSH* for incomming connections

`sudo ufw allow ssh`

# Done.
